# CSS への橋渡し

前回は HTML の基本的な文法を学びました。軽くまとめると、「ここは見出しです！」「ここは段落です！」みたいな感じで、文章の構造を明示していけば良いということになります。
今回は次に学ぶ CSS への橋渡しとして、class と id 、block 要素と inline 要素 について学習したいと思います。

## class と id

たとえば、前回の内容で皆さんは HTML 文書に見出しを入れることが可能になったと思います。しかし、デフォルトでは見出しと言っても文字が大きくなるだけで、あまり目立たないと思います。そこで、以下のように考えたとしましょう。

> 見出しは色を変えたり背景色をつけたりして目立たせたい。特に重要な見出しは他の見出しと色を変えてさらに目立たせたい。

これを実現するには、見出しを区別する必要があります。そのときに用いるのが class や id という「属性」なのです。たとえば、以下のように書くことができます。

```html
<h3 id="important">これは大事な見出し</h3>
<h3 class="normal">これは普通の見出し</h3>
```

このようにすることによって、「important という id がついた h3 要素」のように区別することができるのです。

### class と id の違い

区別するだけでよければ、なんで class と id などというものが存在するのか？どちらかで良いのではないか？ということになりますよね。そこで、class と id の違いについても説明します。

端的にいうと、「**class は何回でも使っていいけど id は一度きり**」ということになります。

詳しく説明すると、同じ class をもつ要素は 1 ページにいくつあっても良いが、同じ id をもつ要素は 1 ページに複数あってはならない、ということです。なんとなく理解できたでしょうか？

<Callout>
    それなら id なんかいらないのでは？と思う人もいるかもしれませんが、id と class
    をうまく使い分けることで**変更に対して柔軟な**サイトを作ることができます。 もし全てが class
    で記述されていたとしたら、1
    箇所のみを修正しようとしても複数箇所に影響してしまう可能性があるのです。
</Callout>

## block 要素と inline 要素

次に紹介するのは block 要素と inline 要素という概念です。

<Callout type="info">
以前はもっと重要な概念として、 HTML タグを学習するときにセットで学習すべきものだったと思うのですが、現在では CSS によって変更が可能なため、後回しにしました。
だからといってデフォルトを無視して良いかというとそういうわけでもないと思うので、概念を押さえ、それぞれの要素と結びつけてもらえると良いと思います。

</Callout>

まずは皆さんに手を動かしてもらって、block 要素と inline 要素の違いを体感してもらいたいと思います。以下のようなファイルを用意してください。

```html filename="index.html"
<!doctype html>
<html lang="ja">
    <head>
        <meta charset="utf-8" />
        <meta name="viewport" content="width=device-width, initial-scale=1.0" />
        <title>Document</title>
    </head>
    <body>
        <div style="background-color: red">これはdivです</div>
        <span style="background-color: lightblue">これはspanです</span>
    </body>
</html>
```

これをブラウザで開いてみてください。どのように表示されたでしょうか？

この画像のようになるはずです。つまり、横にめいっぱい広がるか、中身に合わせて伸び縮みするかの差が出る、というわけです。これは div が block 要素であり、 span が inline 要素 であることから生じる差です。

このように、block 要素と inline 要素では違いがあります。さらにその中間として inline-block というものもあります。長々と説明を書いても分かりづらいので、こちらのページを紹介します。とても分かりやすくまとめられているので、これを見て理解しましょう。

初期値として、`div`・`p`・`ul`・`h1`〜`h6` は block 、 `span`・`a`・`img` は inline が設定されているようです。繰り返しにはなりますが、これらは（一応）CSS で変更が可能です。

## CSS への導入

先ほどの例で、`<div style="background-color: red">{:html}`のように書いた箇所があると思います。このように、色や見た目を変化させるのに使うのが CSS（**C**ascading **S**tyle **S**heet）です。
次回も復習すると思いますが、CSS の側で class を指定するときは `.` を、id を指定するときは `#` をつけます。たとえば、

```html
<h3 id="important">これは大事な見出し</h3>
<h3 class="normal">これは普通の見出し</h3>
```

を指定するときは、

```css
h3#important
h3.normal
```

のように書けば良いことになります。
