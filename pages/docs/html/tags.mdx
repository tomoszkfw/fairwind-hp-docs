# HTML の文法

ここからは、本格的に HTML について学んでいきます。ただの文章だったものがどんどん変化していく様子を楽しんでみてください。

## HTML の文法

前回作った HTML のファイルの中で、たとえば `<title>Document</title>{:html}` という表現が出てきていたと思います。
このように、HTML の文法の特徴はタグでコンテンツを挟んだ「 **要素** 」からなることです。文法を形式的に書くと以下のようになります。

```html
<要素名 属性名=値>コンテンツ</要素名>
```

ここで、属性や子要素は 0 個以上であることに注意しておきましょう。要素が子要素を持ちうるからこそ、HTML の構造は全体として木構造になるのです。
また、子要素がないようなタグについては、 `<要素名 属性名=値 />{:html}` だけで書き表すことができる場合もあります。

## 主な HTML タグの一覧

ここからは、主なタグを列挙してみたいと思います。よく使われる、という基準で選んだもので、これが全てではありません。必要に応じて自分で調べてみましょう。たとえば[こちらのページ](https://developer.mozilla.org/ja/docs/Web/HTML/Element)などは参考になると思います。

-   `<h*>{:html}` タグ（ **h**eading の略）
    -   見出しを表します。
    -   見出しの階層に応じて h1〜h6 までの 6 段階があります。しかし、実際には h6 タグまで使われることは稀です。見出しが 6 段階もある複雑なページなら、ファイルを分けるべきだと考えられるからです。
-   `<p>{:html}` タグ（ **p**aragraph の略）
    -   段落を表します。
    -   標準では、段落と段落の間は間隔が少し開くはずです。 WordPress で普通にエンターキーを押した時の改行はこれです。
-   `<div>{:html}` タグ（ **div**ision の略）
    -   何もしなければ「無色透明な枠」です。
    -   横に目いっぱい広がっている、と思っておいてください。
-   `<span>{:html}` タグ
    -   「無色透明な枠 その２」です。
    -   横幅が中身に合わせて変わる、と思っておいてください。

<Callout>
    これは次回の内容（block 要素と inline 要素）を学ぶと理解が深まると思います。
</Callout>

-   `<em>{:html}` タグ（ **em**phasize の略） / `<strong>{:html}` タグ
    -   どちらも強調したい要素に使います。
    -   何もしなければ、`<em>{:html}` タグは _Italic_ に、`<strong>{:html}` タグは **太字** になるはずです。
-   `<ul>{:html}` タグ（**u**nordered **l**ist の略）/ `<ol>{:html}` タグ（**o**rdered **l**ist の略）/ `<li>{:html}` タグ（**l**ist **i**tem の略）
    -   箇条書きのリストを作るために使います。
    -   `<ul>{:html}` タグは番号がない箇条書き、`<ol>{:html}` タグは番号付きの箇条書きを表します。
    -   どちらのタグも `<li>{:html}` 要素を子要素にとります。箇条書きされる各要素は `<li>{:html}` タグのコンテンツです。
-   `<img>{:html}` タグ（**im**a**g**e の略）
    -   画像を挿入するために使います。
    -   src 属性（source attribute）の値として、画像ファイルへのパスを指定します。
-   `<br>{:html}` タグ（**br**eak の略）
    -   強制的にその位置で改行します。
    -   WordPress での `Shift + Enter` に相当します。
-   `<a>{:html}` タグ（**a**nchor の略）
    -   多くの場合、href（hypertext reference）属性の値としてリンク先の URL （**U**niform **R**esource **L**ocator）を指定することによってリンクを作成します。
    -   例 : [FairWind 公式 HP](https://fairwind-ut.com/)
-   `<button>{:html}` タグ
    -   その名の通り、ボタンを作成します。
    -   ただし、他のページに遷移するようなボタンは `<a>{:html}` タグのデザインを CSS で装飾して作成し、 `<button>{:html}` タグは同じページの中でダイアログを表示するときなどに使われる印象があります。
        -   新歓サイトの使い方を参照してみると理解できると思います。`<button>{:html}` タグが[企画ページ](https://welcome.fairwind-ut.com/activity/event)の上部に見られる一方、[担当ページ](https://welcome.fairwind-ut.com/activity/section)の各担当へのリンクは
            `<a>{:html}` タグで作成されているはずです。
-   `<table>{:html}` タグ
    -   表を作るのに使われます。
    -   そのため、縦横に規則的に並べたい場合に応用されることがありますが、少なくとも新歓サイトを作る過程では一度も見なかった気がします。
        -   この例については[こちら](https://learn.utcode.net/docs/trial-session/html/#問題-2-時間が余った場合)を参照してください。
